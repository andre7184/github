Bem-vindo ao Meu Serviço

Aqui você pode gerenciar seus repositórios do GitHub diretamente do nosso serviço. Utilize o menu para navegar entre as opções disponíveis:

ATENCAO: Para clonar um repositório do GitHub para o nosso servidor ele tem que está publico!

  • Meu GitHub: Acesse suas informações e repositórios do GitHub.
    1. Botao Clonar Repositório: Clone um novo repositório do GitHub para o nosso servidor.
    2. Botao Ver Diretório: Acesse seu diretório já clonado.
  • Repositórios Clonados: Veja a lista de repositórios que você clonou mostrados como Diretórios.
    1. Botao Atualizar: Atualize um Repositório clonado (Busca as mudanças da branch main do repositório remoto origin e as integra na branch atual do repositório local).
    2. Botao Remover: Remove um Repositorio clonado.(caso seu repositorio do github trocou de nome ou nao existe mais, voce deve remover esse diretorio para criar outro com o nome correto) clonado.
    3. Link Acessar Diretório: Acesse o diretório clonado.(para isso seu drepositorio deve possuir arquivo index ou home, html ou php)
  • Meus Dados: Gerencie suas informações pessoais.
  • Logout: Saia da sua conta.

COMO TORNAR UM REPOSITÓRIO PUBLICO

  • Acesse o GitHub:
    • Vá para github.com e faça login na sua conta.
  • Navegue até o Repositório:
    • No canto superior direito, clique no ícone do seu perfil e selecione "Your repositories" (Seus repositórios).
    • Clique no repositório que você deseja tornar público.
  • Acesse as Configurações do Repositório:
    • No menu do repositório, clique em "Settings" (Configurações).
  • Mude a Visibilidade do Repositório:
    • Role a página para baixo até encontrar a seção "Danger Zone" (Zona de Perigo).
    • Clique em "Change repository visibility" (Alterar visibilidade do repositório).
  • Confirme a Alteração:
    • Uma janela pop-up aparecerá. Selecione "Make public" (Tornar público).
    • Digite o nome do repositório para confirmar a alteração.
    • Clique em "I understand, change repository visibility" (Eu entendo, alterar visibilidade do repositório).
  • Verificação de Segurança:
    • Dependendo das suas configurações de segurança, você pode precisar inserir sua senha ou passar por uma verificação de segurança adicional.
